# mirkocasu.github.io

Source code for my personal academic website.

Live site: https://mirkocasu.github.io/

This website is built with Jekyll and based on the
[al-folio](https://github.com/alshedivat/al-folio) theme, with customizations for content, layout, styling, and multilingual pages.

## Overview

The site includes:
- a short bio and academic background
- publications and news
- project and research-related pages
- selected personal links

## Tech stack

- Jekyll
- GitHub Pages
- al-folio
- HTML / SCSS / JavaScript / Liquid

## Local development

To run the site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
